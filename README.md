# 개인 스케줄러 APK

개인용 Android 스케줄러 MVP입니다.

## 기능

- 스케줄 카테고리 추가/삭제
- 카테고리별 상세 스케줄 추가/삭제
- 상세 스케줄의 내용, 횟수/분량, 반복 요일 설정
- 오늘 날짜 기준 할 일 미리보기
- 홈 화면 위젯 추가 시 표시할 카테고리 선택
- 선택한 카테고리의 오늘 할 일을 위젯에 표시
- 백업 파일 내보내기/복원
- 백업 파일은 6자리 난수 코드로 AES-GCM 암호화

## 백업 방식

서버 없이 동작합니다.

1. 기존 폰에서 `백업 내보내기`를 누릅니다.
2. 생성된 JSON 백업 파일을 새 폰으로 옮깁니다.
3. 앱이 보여주는 6자리 복원 코드를 따로 적어둡니다.
4. 새 폰에서 `백업 복원`을 누르고 파일과 코드를 입력합니다.

코드만으로 자동 전송하려면 별도 서버나 근거리 전송 기능이 필요합니다. 이 MVP는 개인용으로 관리 부담이 적은 `파일 + 코드` 방식을 사용합니다.

## 빌드

현재 저장소에는 Gradle Wrapper가 없습니다. Android Studio에서 열면 Gradle 동기화 후 빌드할 수 있습니다.

1. Android Studio에서 이 폴더를 엽니다.
2. Gradle sync를 실행합니다.
3. `Build > Build Bundle(s) / APK(s) > Build APK(s)`를 실행합니다.

이 셸 환경에는 `gradle`과 `ANDROID_HOME`이 잡혀 있지 않아 여기서 APK 빌드 검증은 하지 못했습니다.
